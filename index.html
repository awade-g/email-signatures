Instructions for Gmail:

  1. Copy the signature code from the file
  2. Go to Gmail Settings → See all settings → General → Signature
  3. Click "Create new" signature
  4. Click the three dots menu → "Edit HTML"
  5. Paste the code
  6. Click "Save Changes" at the bottom
